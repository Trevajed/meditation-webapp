## ✦ PHASE 2 — NOTION DATABASE STRUCTURE

### Database: Etherion Chakra Tracks

**Field Schema**:

| Field Name | Type | Purpose |
| --- | --- | --- |
| Chakra Name | Select | Root → Crown classification |
| Track Title | Title | Display name (e.g., "Crimson Ground") |
| Audio File Link | File | Media upload/embed |
| Chakra Frequency | Number | Hz value (396, 417, 528, etc.) |
| Resonance Description | Text | Mythic + healing context |
| Glyph Image | File | Light language sigil (PNG/SVG) |
| Sigil Image | File | Chakra-specific geometric glyph |
| Resonance Code(s) | Text | Frequency codes ("441 / 528 / 963") |
| Download CTA Text | Text | Button copy (e.g., "Receive the Root") |
| Scroll Visual | File | Hero image for chakra page |

- Frequency specifications
- Instrumentation details

**✹ Glyph Meaning**

- Light language interpretation
- Geometric significance

**✹ Resonance Impact**

- Expected somatic experience

- Integration guidance

**✹ Scroll Download**

- Printable PDF version
