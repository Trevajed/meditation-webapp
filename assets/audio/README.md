## Audio Specifications

### Recommended Format
- **Format**: MP3
- **Bitrate**: 128 kbps (good quality, reasonable file size)
- **Sample Rate**: 44.1 kHz
- **Channels**: Stereo

### File Size Guidelines
- **Sample (30-60s)**: ~500 KB - 1 MB
- **Full Track (10-15 min)**: ~10-20 MB
